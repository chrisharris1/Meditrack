RELATIONSHIPS TO CREATE:
=======================

1. User --||--|| Patient
   Relationship: "has profile"
   Type: One-to-One (1:1)
   User.userId connects to Patient.userId

2. User --||--o{ Appointment
   Relationship: "creates"
   Type: One-to-Many (1:N)
   User.userId connects to Appointment.userId

3. Patient --||--o{ Appointment
   Relationship: "schedules"
   Type: One-to-Many (1:N)
   Patient.patientId connects to Appointment.patientId

4. Doctor --||--o{ Appointment (if Doctor entity is created)
   Relationship: "attends"
   Type: One-to-Many (1:N)
   Doctor.name connects to Appointment.primaryPhysician

STARTML CREATION STEPS:
======================

1. Open StarUML
2. Create New Project → Data Model → Entity Relationship Diagram
3. Add entities:
   - Right-click → Add → Entity
   - Name each entity (User, Patient, Appointment, Doctor)
4. Add attributes to each entity:
   - Right-click on entity → Add → Column
   - Set data types, constraints (PK, FK, NOT NULL, etc.)
5. Create relationships:
   - Use "Relationship" tool to connect entities
   - Set cardinality (1:1, 1:N, N:M)
   - Add relationship names
6. Format and layout:
   - Arrange entities for clear visibility
   - Use colors to distinguish entity types
   - Add title and legend

ENUMS TO DEFINE:
================
- Gender: 'male', 'female', 'other'
- Status: 'pending', 'scheduled', 'cancelled'

=======================
